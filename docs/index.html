专为《鹅鸭杀》桌游设计的辅助工具，帮你追踪玩家路径、记录会议信息、统计阵营数据，还支持语音输入和 AI 推理分析。

📸 界面预览

配置人数、地图、阵营比例，选择本局明牌角色。

可视化地图追踪路径，记录每轮移动和目击信息。

玩家卡片记录阵营、角色、可信度，阵营统计一目了然。

✨ 核心功能

🗺️

地图路径追踪

支持老妈鹅飞船、鹅教堂两张地图，可视化记录每轮玩家移动路径，快速定位可疑行为。

📋

会议信息记录

为每位玩家标记阵营、角色、可信度，支持目击记录和备注，会议时一目了然。

📊

阵营统计面板

实时统计鹅/鸭/中立阵营人数，明牌角色、跳出角色、未知槽位一栏显示，死亡角色自动标红。

🔍

角色快速搜索

支持中文名称和拼音首字母搜索，快速找到目标角色，告别翻列表的烦恼。

🎙️

语音控制输入

按住空格键说话，自动识别角色名和地点编号。初始化页说角色名自动勾选，地图页说「X号 地点」自动记录目击，解放双手。

🧠

AI 推理助手 NEW

会议阶段点击「AI帮我捋一下」，接入 DeepSeek-V3 大模型，自动整合当前局所有信息，输出矛盾点、可疑玩家、阵营推算和一句建议，帮你快速理清思路。

💾

本局数据导出

一键导出本局复盘 .txt 文件，包含玩家信息、路径记录、阵营统计，方便赛后回顾。

🚀 如何使用

1

下载项目

点击上方「下载 ZIP」按钮，或前往 GitHub 仓库下载最新版本。
（访问 GitHub 需要梯子，下载后完全离线可用）

2

解压文件

将下载的 .zip 文件解压到任意文件夹。

3

打开工具

双击文件夹内的 index.html，用浏览器打开即可使用，无需安装任何软件。

4

开始游戏

配置人数、地图、阵营数量，选择明牌角色，点击「开始游戏」即可进入记录界面。
游戏中按住空格键说话，自动识别角色名和目击信息。

5

AI 推理分析（可选）

会议阶段点击右上角 ⚙️ AI设置 填入硅基流动 API Key，再点击「🧠 AI帮我捋一下」即可获得 AI 推理摘要。API Key 仅存储在本地，不会上传。

⚠️ 关于访问 GitHub 国内访问 GitHub 可能需要梯子。如果无法下载，可以联系分享者直接获取文件包。工具本身完全离线运行，下载后无需联网。
