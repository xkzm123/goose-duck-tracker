# 鹅鸭杀记牌助手

面向《鹅鸭杀》的浏览器端记牌小工具：配置人数与地图、记录路径与目击、会议阶段整理信息，并可选接入 AI 辅助推理与语音识别。

## 功能概览

- **初始化**：人数、地图、阵营人数、我的角色、明牌角色  
- **游戏阶段**：地图路径、目击记录、语音记录（可选阿里云 / Chrome）  
- **会议阶段**：玩家卡片、阵营统计、可选 AI 分析  
- **其他**：本局导出、横屏体验优化、打赏入口（可选微信小程序 `tip` 路径跳转）

## 本地预览

仓库根目录为静态站点，任选一种方式即可：

```bash
# Python 3（推荐：带禁用缓存响应头）
python server.py
# 浏览器打开 http://localhost:3000
```

若无 Python，可用任意静态服务器（例如 `npx serve .`）打开根目录。

## 部署

根目录已配置 [Netlify](https://www.netlify.com/) 发布（见 `netlify.toml` 的 `publish = "."`），将仓库连接 Netlify 或使用 CLI 部署即可。

## 目录结构（简要）

| 路径 | 说明 |
|------|------|
| `index.html` | 入口页面 |
| `css/styles.css` | 样式 |
| `js/` | 逻辑：`app.js` 主流程，`phase1/2/3`，`state.js`，`data.js`，`ai.js` 等 |
| `images/` | 静态资源（如打赏收款码，可按需替换或从版本库中排除） |

## AI 与语音

在页面 **AI 设置** 中配置：硅基流动 API Key、语音识别方式（Chrome / 阿里云）等；密钥仅保存在浏览器本地。
